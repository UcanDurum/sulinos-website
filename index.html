A Wild Ass

SulinOS is an independent, fast, minimal, general purpose, Inary and OpenRC based free open source Sulin + GNU/Linux distro made by the community for Linux users who want to compile their apps easily.

Download Source


Simplicity

SulinOS is simple because it uses Sulin software, a lightweight alternative to GNU software. Sulin is so lightweight as it is only 100mb. It is very minimal as it follows the UNIX philosophy of 'do one thing and do it well'. Also it doesn't have systemd so regular stuff for developers and regular users such as editing init scripts, compiling packages and booting.



Security

SulinOS is secure because every process requires authentication and wireless connections are held with SSL. Also every compiled package are approved by the community before being published to public.

Inary

Inary is the custom package manager of SulinOS which is forked from Pisi and rewritten in Python 3. Its much more secure and fast predecessor and rivals thanks to the recent improvements and minimization of the source. Also it supports delta packages.





OpenRC

Instead of crappy systemd, SulinOS comes with OpenRC built-in, which is a dependency-based light init system that works with the system-provided init program, normally /sbin/init.
